# Runniac

Runniac es una red social para corredores populares en la que podrán realizar búsquedas de carreras e intercambiar información sobre las mismas con otros usuarios.

El proyecto nace con la intención de dar servicio a una audiencia creciente, y para ofrecer un lugar en el que los usuarios dispongan de un catálogo de carreras completo y exhaustivo con una experiencia de usuario ágil y dinámica, tanto desde equipos personales como de dispositivos móviles.

Además del desarrollo completo, el proyecto contempla la publicación en una plataforma escalable, accesible a través de Internet, y mediante un proceso de despliegue simple, normalizado y siguiendo los principios de la integración continua.

## Estado del proyecto

Actualmente finalizado el desarrollo tras 7 sprints

## Tecnologías empleadas

### Cliente: 
* AngularJS 
* Twitter Bootstrap para el layout y diseño responsive 

### Servidor: 
* WebAPI 2.1 (API Rest)
* ASP.NET MVC 5 
* Autofac (inyección de dependencias) 
* Automapper (mapeo de entidades)
* MSTest (tests unitarios)
* Moq (creación de Mocks )
* Tests funcionales (Specflow y Selenium)

### Base de datos 
* MongoDB 
* SQL Server
